# Typography

Typography tokens define the type scale, font weights, and line heights used throughout the design system.

## Font Family

The design system uses **Inter** as the primary typeface for both display and body text.

```css
font-family: 'Inter', system-ui, sans-serif;
```

## Usage

```jsx
// Display text
<h1 className="text-display-xl font-semibold">Heading</h1>
<h2 className="text-display-md font-semibold">Subheading</h2>

// Body text
<p className="text-md font-regular">Body text</p>
<p className="text-sm font-medium">Small text</p>

// Combining size and weight
<span className="text-lg font-semibold">Label</span>
```
