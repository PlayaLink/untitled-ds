# Border Radius

Border radius tokens create consistent rounded corners across all components.

## Usage

```jsx
// Apply border radius
<div className="rounded-none" />  // 0px
<div className="rounded-xs" />    // 4px
<div className="rounded-sm" />    // 6px
<div className="rounded-md" />    // 8px
<div className="rounded-lg" />    // 10px
<div className="rounded-xl" />    // 12px
<div className="rounded-2xl" />   // 16px
<div className="rounded-full" />  // 9999px (circle/pill)

// Per-corner radius
<div className="rounded-t-lg" />  // top corners only
<div className="rounded-r-xl" />  // right corners only
<div className="rounded-tl-md" /> // top-left only
```

## Common Patterns

| Use Case | Token | Value |
|----------|-------|-------|
| Sharp corners | `none` | 0px |
| Subtle rounding | `xxs`, `xs` | 2px, 4px |
| Buttons, inputs | `lg` | 10px |
| Cards, modals | `xl`, `2xl` | 12px, 16px |
| Large containers | `3xl`, `4xl` | 20px, 24px |
| Pills, avatars | `full` | 9999px |
