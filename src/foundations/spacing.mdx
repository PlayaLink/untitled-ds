# Spacing

Spacing tokens provide consistent rhythm and visual hierarchy throughout the design system. All spacing values are based on a 4px grid.

## Usage

Spacing tokens can be used with padding, margin, gap, and dimension utilities:

```jsx
// Padding
<div className="p-4" />      // 16px all sides
<div className="px-6 py-4" /> // 24px horizontal, 16px vertical
<div className="pt-8" />      // 32px top only

// Margin
<div className="m-2" />       // 8px all sides
<div className="mt-4 mb-6" /> // 16px top, 24px bottom
<div className="mx-auto" />   // auto horizontal centering

// Gap (flexbox/grid)
<div className="flex gap-4" />  // 16px gap between items
<div className="grid gap-6" /> // 24px gap between grid items

// Dimensions
<div className="w-64 h-32" /> // 256px width, 128px height
```

## Common Patterns

| Use Case | Token | Value |
|----------|-------|-------|
| Tight spacing (icons, badges) | 1, 1.5, 2 | 4px, 6px, 8px |
| Component internal padding | 2, 2.5, 3 | 8px, 10px, 12px |
| Standard component padding | 3, 4 | 12px, 16px |
| Section spacing | 6, 8 | 24px, 32px |
| Large section spacing | 12, 16 | 48px, 64px |
| Page sections | 20, 24 | 80px, 96px |
