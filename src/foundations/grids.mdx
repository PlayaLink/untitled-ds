# Grids & Layout

Layout tokens define container widths, breakpoints, and grid systems for consistent page structure.

## Usage

```jsx
// Container with max width
<div className="max-w-container-desktop mx-auto px-4">
  {/* Content */}
</div>

// Fixed width containers
<div className="w-sm" />   // 480px
<div className="w-lg" />   // 640px
<div className="w-2xl" />  // 1024px

// 12-column grid
<div className="grid grid-cols-12 gap-6">
  <div className="col-span-8">Main content</div>
  <div className="col-span-4">Sidebar</div>
</div>

// Responsive grid
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <div>Card 1</div>
  <div>Card 2</div>
  <div>Card 3</div>
</div>
```

## Common Grid Patterns

### Two Column (8/4)
```jsx
<div className="grid grid-cols-12 gap-6">
  <main className="col-span-12 lg:col-span-8">Content</main>
  <aside className="col-span-12 lg:col-span-4">Sidebar</aside>
</div>
```

### Three Column Equal
```jsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-6">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>
```

### Card Grid
```jsx
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-4">
  {cards.map(card => <Card key={card.id} {...card} />)}
</div>
```
